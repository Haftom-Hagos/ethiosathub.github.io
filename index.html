Data

This section provides administrative boundary datasets of Ethiopia in GeoPackage format. The datasets are organized into different administrative levels:

  • Level 0 – National boundary of Ethiopia
  • Level 1 – Regional boundaries
  • Level 2 – Zonal boundaries
  • Level 3 – Woreda (district) boundaries
  • Level 4 – Kebele (local administrative unit) boundaries

Each dataset is available in two coordinate reference systems:

  • Geographic Coordinate System (GCS): EPSG:4326 – WGS 84
  • Projected Coordinate System (PCS): EPSG:32637 – WGS 84 / UTM Zone 37N

The GCS version is useful for web mapping and global datasets, while the PCS version is recommended for spatial analysis and distance/area measurements within Ethiopia. These datasets are suitable for GIS applications, research, and visualization projects related to Ethiopia’s administrative divisions.
